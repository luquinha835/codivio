# 💻 Codivio

🚀 **Codivio** é um editor de código online feito em **HTML** que permite testar projetos web diretamente no navegador.

Ele funciona como uma **mini IDE estilo VS Code**, com editor, preview e terminal integrado.

---

## ✨ Recursos

* 🎨 **Editor com cores estilo VS Code**
* 🌈 **Tema hacker**
* 🧩 Suporte para linguagens:

  * HTML
  * CSS
  * JavaScript
  * Python 
  * C 
* 🖥 **Preview da página em tempo real**
* 💻 **Terminal integrado**
* ⚡ **Atalhos de teclado**
* 💾 **Salvar projetos**
* 📂 **Abrir arquivos**
* 📱 Funciona em **computador e celular**

---

## ⌨️ Atalhos

| Atalho       | Função          |
| ------------ | --------------- |
| Ctrl + Enter | Executar código |
| Ctrl + S     | Salvar projeto  |

## 🚀 Como usar

1. Baixe o arquivo **index.html**
2. Abra no navegador
3. Comece a escrever seu código
4. Clique em **Run** ou pressione **Ctrl + Enter**

## 🌍 Objetivo

O objetivo do **Codivio** é criar uma **IDE simples no navegador** para testar projetos rapidamente sem precisar instalar programas.
